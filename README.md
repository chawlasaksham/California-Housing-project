# California Housing Price Prediction

This project focuses on predicting **median house prices in California** using machine learning techniques.  
Multiple regression models are trained and compared to evaluate their performance on the California Housing dataset.

The project is implemented using **Python and Jupyter Notebook**, with proper data preprocessing, model training, evaluation, and prediction on new input data.
